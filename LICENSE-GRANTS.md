# Additional Use Grants

This file defines the Additional Use Grant for the sundae-sync-v2 Licensed Work
under the Business Source License 1.1.

## Grant Scope

The following entities and projects are granted the right to use sundae-sync-v2
in production, subject to the terms and conditions specified below.

## Granted Projects

### Format

Each grant entry includes:

- **Project Name**: The name of the project or entity
- **Repository**: Link to the project repository (if public)
- **Grant Date**: Date when the grant was issued
- **Permitted Use**: Description of the permitted production use
- **Restrictions**: Any specific restrictions or conditions
- **Expiration**: Grant expiration date (if applicable)

## Requesting a Grant

To request an Additional Use Grant for production use of sundae-sync-v2,
please contact:

**SundaeSwap Labs, Inc.**
Email: contact@sundaeswap.finance
Website: https://sundae.fi

Include in your request:

1. Project name and description
2. Intended use case for sundae-sync-v2
3. Expected deployment scale
4. Organization/individual details
5. Project repository (if applicable)

SundaeSwap Labs reserves the right to:

- Issue new grants at its discretion
- Modify or revoke grants for cause (breach of terms)
- Update grant terms for new applicants
